## 📊 Project Overview
An interactive **Power BI dashboard** analyzing e-commerce sales data. This dashboard provides key business insights including revenue trends, top products, customer behavior, and geographical sales distribution.

## 🔍 Key Metrics
| Metric | Value |
|--------|-------|
| **Total Revenue** | $8.89M |
| **Total Customers** | 392.7K |
| **Total Orders** | 22,014 |

## 📈 Key Insights
- **Revenue Growth:** Significant increase from 2010 to 2011
- **Top Market:** United Kingdom dominates with over 50% of total sales
- **Best-Selling Product:** PAPER CRAFT leads in both revenue and quantity
- **Top Customer:** Customer ID ~17850 spends ~$280K
- **Product Demand:** Clear patterns in top 5 products by quantity

## 📊 Dashboard Features
| Feature | Description |
|---------|-------------|
| **KPI Cards** | Total Revenue, Customers, Orders |
| **Revenue Trend** | Year-over-year revenue growth (2010-2011) |
| **Top Products** | Best-selling products by revenue |
| **Quantity Sold** | High-demand products analysis |
| **Country Sales** | Geographical sales distribution |
| **Top Customers** | Highest spending customers |
| **Interactive Filters** | Year and Country slicers |

## 🛠️ Tools Used
- **Power BI Desktop** - Dashboard creation and visualization
- **DAX** - Custom measures and calculations
- **Power Query** - Data transformation and cleaning
